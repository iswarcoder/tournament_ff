# 🎮 H4X ESPORTS – Free Fire Tournament Platform

A modern and responsive web platform to manage **Free Fire tournaments**.
Users can register, submit payment proof, and track their status in real-time.

---

## 🚀 Live Demo

🔗 https://tournament-ff-frontend.onrender.com/frontend/index.html#home

---

## ✨ Features

* 🧾 User Registration (Name, UID, Phone)
* 💳 Payment Submission (Transaction ID + Screenshot)
* 📊 Live Dashboard

  * Total Users
  * Approved Players
  * Pending Requests
* 🔐 Admin Panel for managing entries
* ⚡ Fast performance
* 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js
* Database: MySQL
* Deployment: Render

## ⚙️ How to Run Locally

1. Clone the repo

```
git clone https://github.com/iswarcoder/tournament_ff.git
```

2. Install dependencies

```
npm install
```

3. Start server

```
node server.js
```

4. Open frontend

```
frontend/index.html
```
